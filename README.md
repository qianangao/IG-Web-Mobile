# IG-Web-Mobile
## Environment
### Ubuntu 18.04
### Python 3.6
## Quickstart
### 1. Clone
`git clone https://github.com/qianangao/IG-Web-Mobile.git`
### 2.Install pipenv 
`cd IG-web-mobile/ig-web-mobile`
`pipenv install`
`pipenv shell`
### 3.Run flask app
    `export FLASK_APP=main.py`
    `flask run`
